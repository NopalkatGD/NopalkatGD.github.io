Validar Sudoku con python

Thu Oct 17 2024
  • Leetcode
  • Python
  • Challenge

# Instroducción

En este desafio de leetcode consiste unicamente en validar si una matriz que sirve de Sudoku es resolvible o no retornando True y False, esto es basandose en las reglas de Sudoku tradicional

# Descripción de leetcode

Define si una un tabla de Sudoku de 9x9 utilizando unicamente con las celdas que esten llenas siguiendo las siguientes reglas:

  • cada fila (horizontal) deben tener digitos del 1-9 sin repetirse.
  • cada columna (vertical) deben tener digitos del 1-9 sin repetirse.
  • cada subcuadro de 3x3 deben tener digitos del 1-9 sin repetirse.

# Notas:

  • La tabla de Sudoku (parcialmente lleno) unicamente debe validarse si es posible resolverse, no llenar o resolver el sudoku.
  • Se debe validarse unicamente con las celdas que estén llenas y siguiendo las reglas anteriores.

# Restricciones:

  • El eje Y debe ser de 9.
  • El eje X debe ser de 9.
  • Cada celda deben tener digitos de 1-9 o un punto "."

# Resolución

Primero se hará una pequeña planificación de como validar la matriz y despues codificar mientras se ven resultados.

# Planeación

Para empezar, se hacen funciones para validar subprocesos:

  • Validar listas: se trabajan sobre listas.
  • Validar Ejes: mediante el proceso de validar listas, se validan los ejes y y x.
  • Validar subcuadros: mediante las coordenadas, se aislan cuadros de 3x3 para validar con la primera función.
  • Validar todos los subcuadros: dando saltos de 3 en 3 mediante ciclos, se validan todos los subcuadros dentro del sudoku o matriz
  • Complemetar funciones: en la función principal que da y utiliza leetcode, se implementan las funciones de validar ejes y de validar todos los subcuadros

# Codificación

# Validador de lista

  1. Se declaran las variables con valores por defectos valid = True y lst = []
  2. El ciclo for se toman cada elemento del parametro iterable
  3. Con la funcion element.isnumeric() mediante un if se consideran unicamente los elementos que sean numeros aunque sean de tipo str
  4. Con el ultimo if se validan la existencia del iterable[i](element) en la variable lst; si no se encuentra iterable[i] en lst, se añade iterable[i] a la lista; si se encuentra, rompe el ciclo y cambia la variable valid a False:
  5. Retorna el valor de valid
def valid_list(self,iterable):
    valid  = True
    lst = []
    for element in iterable:
        if element.isnumeric():
            if not element in lst:
                lst.append(element)
            else:
                print(iterable)
                valid = False
                break
    return valid

# Validar Ejes

  1. El primer ciclo for toma las sublistas en la matriz, desde ese punto se puede validar cada lista o eje x, rompiendo el ciclo en caso de que no se valide y retorne False
  2. A partir del segundo ciclo, se toma cada valor valores en vertical o del eje y para despues generar una lista en la variable sublst = [] para que despues validarlo y romperlo en caso de que no cumpla con las reglas.
def valid_ejes(self, board):
    valid = True
    for i in range(0,9):
        if self.valid_list(board[i]):
            sublst = []
            for j in range(0,9):
                sublst.append(board[j][i])
            if not self.valid_list(sublst):
                #print(sublst)
                valid =False
                break
        else:
            print(board[i])
            valid =False
            break
    return valid

# Validar subcuadros

  1. Para iniciar el rango determinado se toman los valores de X y Y se toman de los parametros y1 y x1
  2. Para dar un limite al rango, los valores de y1 y x1, se le suman int(3) y el resultado se guarde en las variables x2 y y2
  3. Mediante dos ciclos for y los los rangos determinados previamente range(x1,x2) y range(y1,y2) para generar coordenadas en las variable i y j
  4. Utilizando las variables i y j, se guardan los valores que se encuentran en celdas determinadas en la variable lst = [] mediante la función lst.append(board[i][j])
  5. utilizando la funcion de validar lista, se retorna si la variable lst es valida o no
def valid_subsquare(self, board, y1=0 , x1=0):
    x2 = x1 +3
    y2 = y1 +3
    lst = []
    for i in range(x1,x2):
        for j in range(y1,y2):
            lst.append(board[i][j])
    return self.valid_list(lst)

# Validar todos los cuadros

  1. Ahora con ciclos while para ir incrementando de 3 en 3, para obtener las coordenadas que entraran como parametros y se usara la funcion para validar subcuadros, esto para retornar False en caso de que haya un subcuadro invalido
def valid_squares(self,board):
    valid = True
    x = 0
    while x < 9:
        y = 0
        while y <9:
            if not self.valid_subsquare(board, y,x):
                valid = False
            y+=3
        x+=3
    return valid

# Implementación de funciones

Dentro de la función que dá Leetcode, se hace una comparativa mediante if utilizando las funciones de validar ejes y de validar todos lo subcuadros, para que en caso de que ambas funciones retornen True, este validará todo el sudoku, en caso contrario incluso si uno retorne True, este igual retornará False

def isValidSudoku(self, board: list[list[str]]) -> bool:
        if self.valid_ejes(board) and self.valid_squares(board):
            return True
        else:
            return False
