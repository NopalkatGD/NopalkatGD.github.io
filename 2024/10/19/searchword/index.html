# Instrucciones

Este reto consiste en buscar una palabra dada dentro de una matriz para validar si existe devolviendo un True o False en casocontrario

# Descripción

De dará una matriz de m x n y una variable de tipo string, devolviendo un True o False si esta palabra existe o no dentro de la matriz.
Para buscar esta palabra debe ser de manera secuencial con la celdas adyacentes a esta, buscando de manera horizontal o vertical, utilizando una unica vez cada letra que conforma la palabra.

# Restricciones

  1. m es el tamaño de la tabla
  2. n el tamaño de cada fila de la tabla
  3. m debe ser mayo a 1
  4. n debe ser menor a 6
  5. la palabra debe ser mayor a 1 pero menor a 15

# Resolución

Este reto es bastande dificil por lo que procederé primero dar una planeación, despues ideas que tuve durante, despues explicar cada parte del código por ceparado y al final dar el código completo.

# Planeación

  1. primero mediante una función, buscar la ubicación de cada letra que compone la palabra incluso si esta se repite dentro de la palabra
  2. Guardar todas las ubicaciones en una lista
  3. Empezar por las coordenadas de la inicial los cuales diré caminos.
  4. Buscar en la lista de coordenadas
  5. Hacer operaciones para buscar una nueva coordenada dentro de una lista de coordenadas

class Solution:
    def exist(self, board: list[list[str]], word: str) -> bool:
        valid = True
        all_cords = []
        lst_no_valid = []
        anterior = []
        for chara in word:
            coords = self.find_ubic(chara, board)
            all_cords.append(coords)
            print(chara, coords)
        for coord in all_cords[0]:
            valid = True
            contador = 1
            posicion = coord
            print(f"\nprobando camino: {posicion}")
            while contador < len(all_cords):
                if contador >= 1:
                    x = posicion[0]
                    y = posicion[1]
