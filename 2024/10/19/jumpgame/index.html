Jump Game con python

Sat Oct 19 2024
  • Leetcode
  • Python
  • Challenge

# Instroducción

En este desafio de leetcode consiste en validar si una lista puede dar los suficientes saltos para llegar al final devolviendo True y False, la distancia del salto es determinado por el valor del elemento que se encuentra en ese momento.

# Descripción de leetcode

Se te dará una lista con valores numericos int(), inicialmente se ubica en la primera posición de la lista 0, y el valor numerico que se encuentra en esa posición determinará la lonjitud del salto, devolviendo True y False si es posibel llegar al final de la lista.

# Resolución

Pareceria dificil, sin embargo mientra se codifica es posible dar una esplicación ya que es muy corto y sencillo.

# Codificación

  1. Se inicializan las variables con variables determinadas donde: valid = True que cambiará dependiendo el valor de la posición actual, position = 0 que determina la posición actual en la lista y cambiará dependiendo el valor de la celda donde se encuentra actualmente y countador = 1 es un seudocontador que determina el valor de a variable position
  2. El ciclo while ejecutará el código hasta que el valor de position sea mayor al tamaño de la lista
  3. El primer if determina que: en caso de que el valor de la posición actual se diferente a 0, el valor de position se le sumaran el valor actual de la celda determinado por nums[position]. En caso contrario: probará si la siguiente posición determinada por el seudo contador countador es igual a 0, detendrá la ejecución y valid será igual a False, en caso contrario, el valor de position será igual a countador. Al final countador se sumará a si mismo un 1
class Solution:
    def canJump(self, nums: list[int]) -> bool:
        valid = True
        position = 0
        countador = 1
        print("Iniciando")
        while position < len(nums):
            if nums[position] != 0:
                print("-",nums[position])
                position += nums[position]
            else:
                if nums[countador] == 0:
                    print("\n[!] Error")
                    valid = False
                    break
                else:
                    print("\nreintentando")
                    position = countador
                countador +=1
        return valid
